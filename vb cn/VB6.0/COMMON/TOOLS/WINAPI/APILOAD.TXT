API 浏览器允许您浏览包含在文本 API 文件或 Jet 数据库中的常数、声明和类型。其中的项目可以复制到剪贴板并随后复制到您的 Visual Basic 代码中。 

API 浏览器包括 Win32 界面的 WIN32API.TXT 文本文件。提供转换成 Jet 数据库的选项是为了提高速度：在打开一个数据库或 .TXT 文件时能更快地显示列表。您只需加载 .TXT 文件并从“文件”菜单中选取“转换...”，就可以将 .TXT 文件转换成 Jet 数据库文件。 

随后将一个 .TXT 文件转换成 Jet 数据库，要获得最佳性能您应随时加载该数据库。

要加载一个 API 文件，应从“文件”菜单中选取“加载文本文件”或“加载数据库文件”。您也可以通过在命令行中使用下列开关来指定所需的文件名，而加载文本文件或数据库文件：

APILOD32.EXE [/T | /D] 文件名 

窗体顶部的下拉框允许您选择浏览常数、声明或类型。通过选取一项并选择“添加”按钮，您可以添加一项到窗体底部的“选定项”列表框中。

要从“选定项”列表框中复制所有项到剪贴板，只需选择“复制”，则该框中的全部内容都将被复制。然后您可以选取模块的“编辑”菜单中“粘贴”项，把它们包括到您的模块中。要从“选定项”列表框中删除一项，选择“删除”按钮。

菜单
------
文件/加载文本文件: 加载文本 API 文件。
文件/加载数据库: 加载由使用“转换文本为数据库”命令创建的 Jet 数据库。 
文件/转换文本为数据库: 创建一个包含 API 信息的 Jet 数据库。加载数据库比加载可比较的文本文件更快。
文件/退出: 关闭 API 浏览器
