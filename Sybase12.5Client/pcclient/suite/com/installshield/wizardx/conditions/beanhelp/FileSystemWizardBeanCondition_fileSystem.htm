The value entered for File System will be compared against the file system of a specified partition on the user's system.

Values

FAT16

Select this value to check if a specified partition on the user's system is FAT16.

FAT32

Select this value to check if a specified partition on the user's system is FAT32.

HPFS

Select this value to check if a specified partition on the user's system is HPFS.

JFS

Select this value to check if a specified partition on the user's system is JFS.

NTFS

Select this value to check if a specified partition on the user's system is NTFS.

The value entered for File System is compared against the result of File Service's getPartitionFormat method. The value entered for Partition is passed into this method.
