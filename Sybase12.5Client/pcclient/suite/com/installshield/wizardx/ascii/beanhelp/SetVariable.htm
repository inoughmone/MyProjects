A Set Variable action prepends, appends, replaces, or deletes a value onto a variable setting within the ASCII file.

A search for the Variable Name, followed by zero or more whitespaces, followed by an equal sign is performed. The last instance found is used.

Based on the specified Location, the Variable Value is
  • prepended or appended to the current value (along with a separator if one was defined), or
  • used to replace the existing value, or
  • deleted from the current value.
