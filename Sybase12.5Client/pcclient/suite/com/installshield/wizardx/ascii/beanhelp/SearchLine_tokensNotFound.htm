Tokens Not Found determines what position, if any, to which to return if no successful match is found.

Values

  • "Start of File" - If no successful match is found, a pointer to the first character on the first line of the file is returned.
  • "End of File" - If no successful match is found, a pointer after the last character on the last line of the file is returned.
  • "No Action" - If no successful match is found, no action is taken. The next action that requires a valid position be set is ignored unless another successful Search Line/Text action is performed first.
