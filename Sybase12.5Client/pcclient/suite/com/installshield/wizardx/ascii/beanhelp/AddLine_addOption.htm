The Add Option determines whether to add text, lines, comment characters, or comment lines to the ASCII file.

Values

  • "Line" - Adds as a line(s). This means that Text Lines will always start on a new line, and will always be followed by a new line character.
  • "Text" - Adds as text. This means that Text Lines will be inserted directly at the specified Location.
  • "Comment Chars" - Adds comment characters as specified by Comment Chars at the specified Location.
  • "Comment Line" - Adds a comment line(s) starting with the comment characters specified by Comment Chars at the specified Location. Each line will start with the specified Comment Chars character.
