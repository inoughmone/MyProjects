The Search Line/Text action searches the ASCII file for occurrences of a specified string, setting a position for use with the next action. It runs before the
  • Add Line/Text/Comment,
  • Delete Line/Text,
  • Delete Lines Between, and
  • Replace Line/Text
actions to set the position in the file for the action.

NOTES
  • If the specified tokens are not found and the Tokens Not Found property is set to "No Action", then the next action that requires a valid position be set is ignored.
  • If multiple Search Line/Text actions are performed in a row, only the last Search Line/Text action's result is used.
