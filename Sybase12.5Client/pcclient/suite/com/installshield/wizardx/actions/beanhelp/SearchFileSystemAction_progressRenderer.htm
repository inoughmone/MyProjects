The Progress Renderer property enables you to select if/how the progress of this action's execution displays to the end user, e.g., whether you want to include a graphic, display billboards, etc.

Click the ellipsis ("...") and select one of the following:

  • "None" — No progress of the installation displays to the end user.
  • "Default Progress Renderer" — Displays the progress of the action using the settings that are defined in the Wizard Install Properties/Default Interface (installer) and Uninstall Properties/Default Interface (for the uninstaller).
  • "GIF Progress Renderer" — Displays a specified GIF file.
