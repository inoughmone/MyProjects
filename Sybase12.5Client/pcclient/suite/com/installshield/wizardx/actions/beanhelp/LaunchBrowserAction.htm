The Launch Browser Wizard Action launches the specified URL using the end user's external browser. While the URL is being launched by the browser, the Wizard continues on to the next panel or action in the Wizard Tree sequence — It does not wait for the browser to be closed before proceeding to the next Wizard action/panel.

WARNING: This Wizard Action only executes if the installer/uninstaller is run in either AWT/GUI or Swing UI modes — It does not execute in either Console or Silent modes. The available UI mode(s) are set in the Installer Tab Install Properties and Uninstaller Tab Uninstall Properties.
