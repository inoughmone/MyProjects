Specify the URL to be launched with the end user's browser, including the protocol to be used. For example:

  • Incorrect — www.installshield.com (will cause an error in the browser)
  • Correct — http://www.installshield.com

You can also use this action to display a local file. A string resolver can also be used. For example:
  • Incorrect — C:/readme.html
  • Correct — file://C:/readme.html
    OR
    file://$P(absoluteInstallLocation)/readme.html
