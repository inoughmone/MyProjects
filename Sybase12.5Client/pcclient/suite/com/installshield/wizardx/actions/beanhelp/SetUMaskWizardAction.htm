The Set UMask Wizard Action provides the ability to set the target system umask for the installer process in order to control the permissions assigned to files and directories created by the installer. This does not affect files and directories created by a Files bean(s) because they explicitly set the permissions at install time regardless of the umask. For other files and directories created at install time, the file mode creation mask (umask) controls the permissions assigned to them.

On systems that do not support the concept of a umask, this bean does nothing.

NOTE: This bean is not supported for use with the Generic Unix Platform Pack.
