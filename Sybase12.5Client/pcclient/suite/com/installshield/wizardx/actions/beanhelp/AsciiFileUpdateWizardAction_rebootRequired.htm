Reboot Required is used to determine whether the system will require a reboot if the ASCII file is modified.

Values
True
The operating system should reboot for the changes in the modified ASCII file to take effect.

False

A reboot of the operating system is not required for the changes in the modified ASCII file to take effect.

If Reboot Required is set to true then the RebootPanel should be included at the end of the Install and Uninstall Wizard Trees.
