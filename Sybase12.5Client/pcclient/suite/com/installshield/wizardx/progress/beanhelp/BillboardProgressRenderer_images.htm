The Billboard images represent the GIF file(s) (including animated GIF(s)) that you want to display either in the Wizard panel or full-screen. How the image(s) displays depends upon the setting of the Display Mode property — "Within Panel" or "Full Screen".

Click the Add button and browse to each GIF file you want displayed. Once added, their display sequence can be modified by highlighting the file name and using the up/down arrow keys.

NOTE: These files cannot be clicked and dragged into the IDE from your file management utility.
