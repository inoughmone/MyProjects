The Include Localized Images property enables you to include localized versions of your billboard images. If the specified image(s) should be localized, then set this property to "True". Otherwise, leave this property set to "False" (default).

If set to "True", then ensure that the correctly named localized graphic files are located in the same directory as the default file. A localized image file must comply with the class naming convention pattern described in Sun's API documentation for Resource Bundles (http://java.sun.com/j2se/1.4.1/docs/api/java/util/ResourceBundle.html), e.g., "billboard_en.gif" (for English), "billboard_es.gif" (for Spanish), etc.

NOTE: The list of localized files to be included in the archive is determined by the languages specified under Language Support in the Build Tab. For a list of supported languages and their associated Java abbreviation, see the MultiPlatform User's Guide, Chapter 4.0 Build Tab/Language Support.
