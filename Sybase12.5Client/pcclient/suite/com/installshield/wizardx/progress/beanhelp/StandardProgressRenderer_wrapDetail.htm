The Wrap Detail property controls whether the detail caption wraps to multiple lines if it is too long to display in one line.

  • "True" — The entire detail caption displays, utilizing multiple lines, if required.
  • "False" — The detail caption is spliced with the missing information replaced with an ellipsis ("...") in order for it to fit on one line, e.g., "/user1/great_product/.../foo.txt".
