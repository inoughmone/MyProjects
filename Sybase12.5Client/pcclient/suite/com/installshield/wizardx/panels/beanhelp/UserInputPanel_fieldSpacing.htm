Specify the amount of space, in pixels, that you want between each field in the panel. The default is "8", which gives the panel a uniform look.

Additional spacing can also be specified for each field by setting their Spacing properties, located in the User Input Field editor dialog box in the Layout Tab:

  • "Above"
  • "Left of"
  • "Below"
  • "Right of"

These field properties give you additional options for customizing the look of the panel.
