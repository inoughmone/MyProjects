Specify the text to display on the panel. To change this text, click the ellipsis ("..."), and the Specify Text dialog box displays.

You can replace the default localized string "$L(com.installshield.wizardx.i18n.WizardXResources, LogoutPanel.text)" with the location of your own localized file, or the exact text that you want to display in the panel. (You can see the exact wording of this default string — See the MultiPlatform User's Guide, Chapter 8.4 — String Resolution & Localization/Localized String Resources.
