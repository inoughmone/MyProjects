The value selected for Restart Wizard determines whether the wizard will be restarted after the user's system reboots.

Values

True
The wizard will be restarted after the system reboot.

False

The wizard will not be restarted after the system reboot.

If the value for Restart Wizard is 'True', then the wizard will restart at the next wizard bean in the wizard tree (i.e., the wizard bean immediately following the Reboot Panel). If the value for Restart Wizard is 'True' and the Reboot Panel is the last wizard bean in the wizard tree, then the system will simply reboot and a wizard restart will not occur.

Note: Restarting the wizard is only supported through the use of native launchers.
