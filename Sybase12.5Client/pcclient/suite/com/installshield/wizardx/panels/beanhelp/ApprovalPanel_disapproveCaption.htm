Specify the caption to display with the disapproval button or selection. For example "I DO NOT agree".

NOTE: The mnemonic <Alt+D> can also be used by the end user to make this selection.

You can set the mnemonic for this property by prepending this property's value with an ampersand ("&"). The first letter of the string becomes the mnemonic and displays underlined. For example:

&No, I Do Not Approve

displays the caption as No, I Do Not Approve, and is selected from the list when the end user types <Ctrl+N>.
