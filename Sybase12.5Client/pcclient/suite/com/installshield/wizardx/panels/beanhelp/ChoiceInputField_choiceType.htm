To select a Choice Type, click the down arrow, and select one of the following types from the drop-down list:

  • "Radio Button List" - Only one selection can be made
  • "Check Box List" - More than one selection can be made
  • "Single Select List" - Only one selection can be made
  • "Dropdown List" - Only one selection can be made
  • "Scrollable Check List" - More than one selection can be made
