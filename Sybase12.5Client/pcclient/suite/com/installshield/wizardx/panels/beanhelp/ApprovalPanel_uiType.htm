Click the down arrow and select the User Interface (UI) convention for the Approval choice: Either "Drop-down List" or "Radio Buttons".

NOTE: If "Drop-down List" is selected, and Selection (property immediately above this one) is set to "Unset", then the first option — "Approve" — displays in the drop-down list as preselected when this panel initially displays. Therefore, it is recommended that if a "Drop-down List" is required as the UI Type, then the Selection property should be set to either "Approve" or "Disapprove". If no preset selection is desired (Selection value is "Unset"), then this property should be set to "Radio Buttons".
