Specify the caption to display with the Approval button or selection. For example, "I agree".

NOTE: The mnemonic <Alt+A> can also be used by the end user to make this selection.

You can set the mnemonic for this property by prepending this property's value with an ampersand ("&"). The first letter of the string becomes the mnemonic and displays underlined. For example:

&Yes, I Approve

displays the caption as Yes, I Approve, and is selected from the list when the end user types <Ctrl+Y>.
