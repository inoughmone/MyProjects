Click the down arrow and select the option that is preselected when this panel initially displays:

  • "Unset" (none)
  • "Approve"
  • "Disapprove"

NOTE: If UI Type is set to "Drop-down List", and this property is set to "Unset", then when this panel initially displays, the first option — "Approve" — displays as preselected in the drop-down list. Therefore, it is recommended that if a "Drop-down List" is required as the UI Type, then this property be set to either "Approve" or "Disapprove". If no preset selection is desired (this property value is "Unset"), then the UI Type property should be set to "Radio Buttons".
