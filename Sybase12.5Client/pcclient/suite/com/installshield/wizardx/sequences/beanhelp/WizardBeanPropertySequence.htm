A Wizard Bean Property Sequence is used to group two or more Wizard beans together and visit them based on the value of a specified Wizard bean property and/or a Condition(s).

NOTE: The following are the situations when all of the Wizard beans grouped within this sequence are not visited at runtime:

  • The specified Property Name does not have a value that matches the string or GNU Regular Expression specified in Property Value.
  • All of the Conditions set for this sequence do not evaluate to "True" at runtime.
